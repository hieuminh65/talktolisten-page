API Documentation

Speech to Text Model

This machine learning model transcribe users' speech to text when they are talking

We will use Azure AI Speech and OpenAI Whisper API , but later will transfer to hosting Whisper AI on our own

Text to Text Model (Large Language Model)

This model processes user inputs, understanding the context and nuances of their text. This model is crucial for ensuring that interactions are not only responsive but also meaningful, providing users with a realistic and engaging conversational experience. Whether it's answering questions, offering suggestions, or engaging in casual banter, this model adapts to the user's conversational style and content, making each interaction uniquely personalized

We will use OpenAI Assistant API to custom the bot and save the conversation history

Text to Speech (Speech Synthesis) Model

This model use the clone voice technologies and converts the bot's text responses into speech, allowing users to hear the bot's voice. We will use llElevenLabs API and later transfer to opensource models: suno-ai/bark, coqui-ai/TTS, CorentinJ/Real-Time-Voice-Cloning

For default bots we will use text to speech model of suno-ai/bark, coqui-ai/TTS, and OpenAI Speech to Text API

Small Text to Text Model (Large Language Model)

This small large language model will only be used to custom the bot based on what user want. It reparaphase user's preference and generate instruction for the bot

We will use OpenAI Chat Completion GPT 3.5 model but latter might switch to opensource model HuggingFaceH4/zephyr-7b-beta for easier fine-tuning
