Documentation

Detailed documentation will be in the github repository

Front-End

The front-end facilitates user interaction, allowing users to log in, register, create and customize their bots with voice and avatar uploads, and engage in conversations. It also displays a variety of default bots

  • Tech Stacks: React Native (React.js, JavaScript), Figma
  • Front-end Github Repository
  • Figma Design

Back-End

The back-end handles requests from the mobile app, managing multiple APIs and leveraging machine learning models for speech-to-text, text-to-text, and text-to-speech conversions

  • Tech Stacks: Python, FastAPI, AWS EC2, AWS S3, Runpod, FireBase
  • Tools: Google Collab
  • API Reference
  • Back-end Github Repository

Database

The database securely stores user data, including profile information, or bot configurations

Tech Stacks: PostgreSQL, AWS RDS
